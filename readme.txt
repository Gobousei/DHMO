献立保存・活用ソフト　con-dateの概要並びに運用計画

１．目的
　con-date（以下、本ソフト）は、カレンダー、データベースを利用し、その日の献立内容・レシピ・材料等を記録・管理することを目的として作成するものである。
 
２．概要
　本ソフトにて、その日作った献立の名前・レシピの画像・材料などを入力・アップロードし、日付・材料・料理名などで分類・記録する。
　作った日付・料理名・材料などからデータベースを検索し、必要であればそのレシピを印刷できるようにする。
　同じ料理名のものを作っている場合、それを検索し、そのレシピについて感想、考察などを入力、記録できる。
 
３．運用計画
　本ソフトは、すべてオフラインでの使用を前提に作成するが、データベースの移行などの際にデータをクラウド上にアップできるよう、データをxlsx形式で保存する。
　又、保存したデータは必要に応じてそれぞれの項目で並び替えを行った後に印刷を行えるよう整備する。
